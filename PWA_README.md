# CODE VED PWA

A fully functional Progressive Web App (PWA) with glassmorphism UI design, offline support, and native app feel.

## 🚀 Features

### PWA Capabilities
- **Installable**: Add to home screen on mobile and desktop
- **Offline Support**: Works without internet connection using Service Worker caching
- **Background Sync**: Queue actions when offline, sync when online
- **Push Notifications**: Ready for push notification support
- **Custom Offline Page**: Beautiful glassmorphism-styled offline/retry page

### Native App Feel
- No browser chrome or URL bar when installed
- Disabled text selection highlights
- Transparent tap/click highlights (`-webkit-tap-highlight-color: transparent`)
- Smooth page transitions
- Gesture handling for native-like interactions
- Safe area insets for notched devices

### Glassmorphism Design
- Semi-transparent backgrounds with backdrop blur
- Subtle borders and soft shadows
- Modern, cohesive aesthetic across all UI elements
- Animated background elements on offline page

## 🛠️ Build & Deploy

### Local Development

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```

2. **Run the build script**
   ```bash
   ./build.sh
   ```

3. **Serve locally**
   ```bash
   npx serve dist/
   ```

4. **Open in browser**
   - Navigate to `http://localhost:3000`
   - Use Chrome DevTools → Application tab to test PWA features

### GitHub Actions Deployment

The included workflow automatically:
- Validates all PWA files
- Builds the distribution package
- Deploys to GitHub Pages
- Optionally deploys to Hugging Face Spaces
- Runs Lighthouse PWA audit

#### Setup Secrets (for HF deployment)
- `HF_TOKEN`: Your Hugging Face API token
- `HF_REPO`: Your Hugging Face Spaces repository name

## 📱 Installation

### Desktop (Chrome/Edge)
1. Open the app in browser
2. Click the install icon in the address bar
3. Click "Install"

### Mobile (Android Chrome)
1. Open the app in Chrome
2. Tap the menu (⋮) → "Add to Home screen"
3. Confirm installation

### Mobile (iOS Safari)
1. Open the app in Safari
2. Tap the Share button
3. Select "Add to Home Screen"
4. Confirm addition

## 🔧 Service Worker Features

- **Cache Strategy**: Network-first with cache fallback
- **Static Assets**: Cached on install
- **External Resources**: Cached on first fetch
- **Offline Detection**: Automatic redirect to offline page
- **Auto-update**: Checks for updates every minute

## 🎨 Customization

### Theme Colors
Edit `manifest.json`:
```json
{
  "theme_color": "#18181b",
  "background_color": "#ffffff"
}
```

### App Name
Edit `manifest.json`:
```json
{
  "name": "Your App Name",
  "short_name": "App"
}
```

### Icons
Replace `logo.png` with your branding, then regenerate sizes:
```bash
python3 generate-icons.py
```
